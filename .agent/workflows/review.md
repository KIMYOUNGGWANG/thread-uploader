# 🔍 /review - 검수 모드 (Gemini Manager)

## 🎯 목적
구현이 완료된 코드를 **초기 기획서와 비교**하여 누락된 기능이나 버그가 없는지 검사합니다.

## 📝 검수 항목
1. **기능 완성도**: PRD의 모든 요구사항이 구현되었는가?
2. **엣지 케이스**: 예외 상황이 적절히 처리되는가?
3. **코드 품질**: 베스트 프랙티스를 따르고 있는가?
4. **보안**: 민감 정보 노출이나 취약점은 없는가?

## 📦 산출물
- **검수 보고서**: PASS/WARNING/FAIL 상태와 상세 피드백
- **수정 요청 목록**: 발견된 이슈와 우선순위

## 🚀 사용 예시
```
구현 끝났어. /review 해줘
```
