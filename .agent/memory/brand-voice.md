# Brand Voice — CosmicPath
> 마지막 업데이트: 2026-06-05

## 브랜드 정보
- **이름:** CosmicPath
- **설명:** 사주 + 점성술 + 타로를 교차해서 지금 막힌 질문의 판정, 근거, 다음 행동을 정리해주는 3단분석 리포트
- **ICP:** 20-35세 여성, 연애·이직·진로·돈·관계 고민을 오래 붙잡고 있고, 막연한 좋은 말보다 지금 움직일지 기다릴지 판단선이 필요한 사람

## 핵심 메시지
질문 하나를 접수하면 사주는 구조, 점성술은 타이밍, 타로는 현재 심리 신호로 나눠 보고 지금 선택 가능한 범위를 좁힌다.

## 브랜드 DNA
- **믿음:** 운세는 불안을 키우는 예언이 아니라, 막힌 질문을 정리하는 도구여야 한다.
- **역할:** CosmicPath는 답을 내려주는 AI가 아니라 조용한 3단분석 접수실이다.
- **핵심 전환:** 미래 확정 → 선택지 좁히기 / 상대 마음 보장 → 내 행동선 정리 / 막연한 위로 → 판정, 근거, 다음 행동

## 판정 언어
- 일반 질문: 움직임 / 대기 / 축소 / 보류
- 관계 질문: 연락 / 대기 / 축소 / 보류
- 판정은 단정이 아니라 지금 선택 가능한 범위를 좁히는 임시 판단선으로 표현한다.
- 상세 리포트는 왜 이 판정인지, 어떤 타이밍인지, 무엇을 피해야 하는지, 어떤 순서로 움직이면 좋은지 설명한다.

## 제품 퍼널
- 1단계: 사용자가 지금 막힌 질문 하나를 접수한다.
- 2단계: 생년월일, 출생지, 출생시간, 양력/음력 정보는 사주와 점성술 보정을 위해 선택적으로 더한다.
- 3단계: 타로는 직접 뽑거나 건너뛸 수 있다.
- 4단계: 질문만으로도 무료 첫 판정은 열 수 있다.
- 5단계: 상세 리포트에서 사주 흐름, 점성 타이밍, 타로 상징을 나란히 놓고 근거와 다음 행동을 정리한다.

## 전환 문장
- 지금 질문을 한 문장으로 접수해보면, 먼저 움직임 / 대기 / 축소 / 보류 중 어디에 가까운지 볼 수 있다.
- 생년 정보와 타로까지 더하면 판정의 근거가 더 선명해진다.
- 첫 판정은 방향을 좁히고, 상세 리포트는 왜 그런지와 어떤 순서로 움직일지를 보여준다.
- 불안해서 계속 검색하는 대신, 질문 하나를 접수하고 행동선을 먼저 정리한다.

## 논란 각도
- **싫어할 사람:** 운세는 무조건 미래를 맞혀야 한다고 믿는 사람, 자극적인 상대 마음 콘텐츠를 원하는 사람
- **50% 혐오 포인트:** 미래를 맞히겠다고 말하지 않는 운세 서비스
- **공공의 적:** 막연한 좋은 말만 주는 운세, 불안을 찌르는 연애 콘텐츠, 결정을 더 미루게 만드는 해석
- **핵심 대립:** "맞힐게요"가 아니라 "지금 질문을 세 가지 근거로 나눠 판단해봅니다"

- **절대 금지:** `#인생_프레임_바꾸기` 같은 합성어나 문장형 해시태그 금지

## 추천 바이럴 템플릿
1. 질문 접수형: "지금 연락해도 될까요?"처럼 실제 질문으로 시작하고 판정 기준을 보여준다.
2. 3단분석형: 사주는 구조, 점성술은 타이밍, 타로는 심리 신호라는 역할 분리를 반복 각인한다.
3. 행동선 정리형: 미래 예언보다 지금 움직임 / 대기 / 축소 / 보류 중 어디에 가까운지 좁혀준다.
4. 불안 반전형: 상대 마음을 맞히기보다 내 행동선을 먼저 정리해야 한다는 관점으로 뒤집는다.
5. 운세 피로 저격형: 좋은 말은 들었는데 뭘 해야 할지 모르겠는 사람을 직접 부른다.

## 질문 카테고리별 각도
- 연애/연락: "상대 마음"을 맞히는 대신 지금 연락 / 대기 / 축소 / 보류 중 어디에 가까운지 정리한다.
- 이직/퇴사: 커리어 운을 단정하지 말고 밀어붙임 / 대기 / 축소 / 보류의 타이밍 감각으로 정리한다.
- 돈/사업: 대박 예언이 아니라 확장 / 보수 / 정리 / 보류 중 현재 리스크 방향을 좁힌다.
- 관계/가족: 누가 맞는지보다 내가 지금 어떤 말과 순서로 움직여야 하는지 본다.
- 하반기/월간 흐름: 좋은 일 예언이 아니라 집중할 영역, 피할 무리수, 먼저 줄일 것을 정리한다.

## 좋은 훅
- 운세를 봐도 결국 뭘 해야 할지 모르겠는 사람에게
- 연락할까 말까 고민이 길어질수록 답이 흐려지는 이유
- 사주만 보면 구조가 보이고, 타로만 보면 감정이 보이고, 점성술까지 보면 타이밍이 보인다
- 좋은 말 듣고 싶은 날이 아니라 결정을 내려야 하는 날 보는 리포트
- 미래 예언보다 필요한 건 지금 움직일지 말지다
- 상대 마음 맞히기보다 내 행동선을 정리하는 게 먼저다
- 질문 하나를 접수하면 세 가지 근거로 나눠 본다
- "올 하반기 어떻게 해야 좀 풀릴까요?"는 운세 질문이 아니라 행동 순서 질문일 수 있다
- "지금 연락해도 될까요?"에 필요한 건 확신보다 판정 기준이다
- 이직운보다 먼저 봐야 하는 건 지금 밀어붙여도 되는 타이밍인지다

## 포스트 구조
- 1문단: 사용자가 실제로 할 법한 질문을 그대로 꺼낸다.
- 2문단: 그 질문이 왜 단순한 예/아니오 문제가 아닌지 짚는다.
- 3문단: 사주 / 점성술 / 타로가 각각 무엇을 보는지 나눈다.
- 4문단: 움직임 / 대기 / 축소 / 보류 중 하나의 판단선으로 정리한다.
- 5문단: 미래 확정이 아니라 다음 행동을 좁히는 리포트라고 닫는다.

## 바이럴 품질 기준
- 첫 줄은 반드시 실제 고민 문장이나 반직관 문장으로 시작한다.
- 본문은 불안을 키우지 않고 판단 기준을 준다.
- 최소 한 번은 사주 / 점성술 / 타로의 역할 분리를 보여준다.
- 마지막은 "맞힌다"가 아니라 "질문을 접수한다", "행동선을 정리한다", "선택지를 좁힌다"로 닫는다.
- 전환이 필요할 때만 첫 판정 / 상세 리포트를 언급하고, 앱 광고처럼 보이는 직접 다운로드 문장은 피한다.

- "상대 마음을 알려준다"
- "미래를 보장한다"
- "AI가 운명을 예측한다"
- "무조건 연락해라"
- "무조건 헤어진다/된다"
- "프리미엄 운세 대박"
- "오라클/데스티니/AI 리포트" 중심 표현
- 공포, 집착, 재회 강박을 자극하는 표현
- "~해야 합니다", "~것 같습니다" 같은 공손체
- 논문 같은 설명체

## 권장 톤
- 차분하고 선명하게
- 신비롭지만 허황되지 않게
- 불안을 찌르지 말고 질문을 정리해주는 느낌
- "당신의 운명을 맞혀드립니다"가 아니라 "지금 질문을 세 가지 근거로 나눠 판단해봅니다"
- 브랜드명은 직접 광고처럼 반복하지 말고, 필요할 때만 자연스럽게 사용한다.
