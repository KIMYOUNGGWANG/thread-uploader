# Office Hours — CosmicPath 마케팅 자동화
> 작성일: 2026-04-15

## 제품 개요

| 항목 | 내용 |
|------|------|
| 서비스명 | CosmicPath |
| 유형 | Freemium 웹 서비스 |
| 내용 | 사주 + 점성술 + 타로 통합 분석 |
| 무료 | 간단한 사주 정보 |
| 유료 | 디테일한 분석 |
| ICP | 20-35세 여성, 연애·취업·진로 고민 |

---

## 현재 상태

- Threads에 하루 5개 자동 포스팅 중
- 바이럴 성과 있음: 29만 조회(로또 상상 + 사주), 9.9만, 2.4만 등
- 그러나 **최근 콘텐츠가 동기부여/자기계발 계열로 drift** → 바이럴 없음
- 포스팅 자동화는 있지만 **성과 데이터 수집 없음**
- **웹사이트 전환 추적 없음** (UTM 없음, 유입 분석 없음)

---

## 핵심 문제 (2가지)

### 문제 1: 콘텐츠 Drift
증명된 바이럴 공식(사주 특화 훅 + 전문 용어 + 참여 유도)에서 콘텐츠가 이탈함.
시스템적으로 공식을 강제할 장치가 없어 AI가 generic 동기부여 글을 생성함.

### 문제 2: 전환 루프 없음
29만 조회가 나와도 웹사이트 유입 → 무료 사용 → 유료 전환으로 이어지는 데이터가 없음.
무엇이 실제로 돈이 되는지 알 수 없음.

---

## 바이럴 공식 (검증됨)

성공한 게시물 패턴:
1. **사주 전문 용어 훅**: "혹시 X보다 센 게 뭔지 알아?" → 궁금증 유발
2. **구체적 사주 개념**: 도화, 홍염, 화개, 진술축미, 천을귀인 등 실제 용어 사용
3. **참여 유도 구조**: 선택지, 1/2 시리즈, 댓글 유도
4. **공감 시나리오**: 로또 상상처럼 타깃의 욕망(연애·돈·직업)과 직결

실패 패턴:
- 사주 없는 동기부여 ("작은 변화", "할 수 있다")
- 긴 에세이형
- 훅 없는 첫 문장

---

## 자동화 목표

```
[콘텐츠 생성] → [품질 검증] → [자동 포스팅] → [성과 수집] → [전략 피드백]
```

1. **콘텐츠 생성**: 검증된 바이럴 공식을 강제하는 프롬프트로 사주 훅 콘텐츠 생성
2. **품질 검증**: 바이럴 공식 준수 여부 자동 체크 (drift 방지)
3. **자동 포스팅**: 기존 threads-uploader 활용
4. **성과 수집**: 조회수·좋아요·댓글 데이터 주기적 수집
5. **전환 추적**: 게시물 → 웹사이트 UTM 파라미터 연결
6. **전략 피드백**: 성과 상위 20% 패턴 분석 → 다음 배치 프롬프트 반영
